# TechHub - Electronics Store

A responsive e-commerce website for an electronics store built with HTML, CSS, and JavaScript only.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Product Catalog**: Browse electronics including smartphones, laptops, tablets, and accessories
- **Product Details**: View detailed specifications and descriptions
- **Shopping Cart**: Add/remove items, update quantities with localStorage persistence
- **Category Filtering**: Filter products by category
- **Product Sorting**: Sort by price and name
- **Modern UI**: Clean, intuitive interface with smooth animations

## Pages

1. **Home Page** (`index.html`)
   - Hero section with call-to-action
   - Featured products showcase
   - Product categories

2. **Products Page** (`products.html`)
   - Complete product listing
   - Category and sorting filters
   - Responsive grid layout

3. **Product Detail Page** (`product-detail.html`)
   - Detailed product information
   - Full specifications
   - Quantity selector
   - Add to cart functionality

4. **Shopping Cart** (`cart.html`)
   - Cart items management
   - Quantity updates
   - Order summary
   - Checkout simulation
